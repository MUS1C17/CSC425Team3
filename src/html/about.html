About This Project

This starter is designed to be clean, accessible, and easy to extend. It uses semantic HTML, a shared design system, and zero dependencies.

Clarity

Readable code, predictable structure, and consistent styling.

Accessibility

Landmarks, focus states, and sensible defaults from the start.

Scalability

Source files live in src/ so you can grow into tooling later.

FAQ

Can I add frameworks later?

Yes. Keep the structure as-is and introduce a bundler (Vite/Webpack) when ready.

How do I deploy?

Any static host works: GitHub Pages, Netlify, Vercel, or a simple web server.

Does the dynamic page require a backend?

No—it's a purely client-side demo using vanilla JavaScript.
